## How to Use

### Dashboard (Home)
- **View recent tests** and scores
- **Quick access CTAs:**
  - "⚡ Start Mock Test" → Begin a full-length mock
  - "Chapter Practice" → Select a chapter
- **Performance metrics** — Quick overview of progress
- **Feature highlights** — Explore all available tools

### Taking a Mock Test
1. Navigate to **Mocks** section
2. Select any mock test
3. Click **"Start test"** button
4. Read each question carefully
5. Select your answer from 4 options
6. **Mark for Review** if uncertain
7. **Navigation:**
   - Use Prev/Next buttons OR
   - Click question numbers in right panel
8. Click **"Submit Test"** when complete
9. View instant results and explanations

### Using Chapter Practice
1. Go to **Chapter Practice**
2. Filter by subject (Accountancy, Business Studies, etc.)
3. Select any chapter card
4. Click to launch chapter mock (50 questions)
5. Same interface as regular mocks
6. Progress automatically tracked

### Using PYQ Bank
1. Navigate to **PYQ Bank**
2. Filter by **year** (2025, 2024, 2023, 2022)
3. Filter by **subject** (Accountancy, Business Studies, Economics)
4. Select any paper
5. Click **"Attempt"**
6. Take the full test (same as mocks)
7. Review explanations after submission

### Studying with Flashcards
1. Go to **Flashcards**
2. Choose mode:
   - **English Vocab** (vocabulary)
   - **Accountancy** (chapter cards)
   - **Business Studies** (topic cards)
3. Select year (2025, 2024, etc.) or "All years"
4. Choose deck
5. **Study mode:**
   - Read question/word
   - Click to reveal answer
   - Rate difficulty (Hard/OK/Easy)
   - Move to next card
6. Continue until deck complete

### Reviewing Analysis
1. Navigate to **Analysis**
2. Filter by type:
   - **ALL** — All tests and chapters
   - **MOCKS** — Full-length tests only
   - **CHAPTERS** — Chapter practice only
3. View metrics:
   - Overall accuracy
   - Subject-wise performance
   - Chapter-wise breakdown
   - Activity heatmap
4. Identify weak topics for focused practice

### Using Question Library
1. Go to **Library**
2. View two tabs:
   - **SAVED** — Questions you bookmarked
   - **ERRORS** — Questions you got wrong
3. **Search & filter:**
   - Search by keyword
   - Filter by subject/difficulty
4. Review full question + explanation
5. Remove questions as you master them

### College Predictor
1. Navigate to **College Predictor**
2. Select program: B.Com (Hons) or B.Com (Pass)
3. Select category: UR, OBC, SC, ST, EWS, or PwBD
4. Adjust score sliders:

## FAQs

### Q: Do I need internet to use it?
**A:** No. The application works completely offline. Download once and use forever.

### Q: Is this the real CUET exam?
**A:** No, this is practice software. The questions are from official CUET papers and follow the same format, but it's not the actual exam. The real exam happens on NTA's portal.

### Q: How many questions are included?
**A:** 
- 21 × 50 = 1,050 PYQ questions (government papers)
- 13 chapters × 50 = 650+ chapter questions
- 500+ questions in NCERT notes
- **Total: 2,200+ questions**

### Q: Can I share my results?
**A:** Yes, you can screenshot your result card or manually share your score. Data is stored locally in your browser.

### Q: What if I lose my progress?
**A:** Progress is saved in your browser's localStorage. If you clear browser data, you'll lose progress. We recommend exporting results before clearing data.

### Q: Can multiple users use the same file?
**A:** Yes, each user can create their own account with their own username/password. Progress is tracked separately per user.

### Q: How is the scoring done?
**A:** 
- Correct answer: +5 marks
- Wrong answer: -1 mark
- Skipped: 0 marks
- Total: 50 × 5 = 250 marks maximum

### Q: Can I retake mocks?
**A:** Yes, unlimited retakes. Take the same mock multiple times to improve.

### Q: How accurate are the explanations?
**A:** All explanations are based on official CUET papers and verified solutions. Each explanation covers:
- Core concept
- Formula/Rule application
- Step-by-step working
- Common mistakes

### Q: Is my data secure?
**A:** Yes. All data is stored **only in your browser** (localStorage). No data is sent to any server or cloud. It's 100% private.

## Tips for Effective Preparation

### Recommended Study Plan

**Week 1-2: Foundation**
- Take 2-3 full mocks to assess baseline
- Review explanations carefully
- Identify weak chapters

**Week 3-4: Targeted Practice**
- Focus on weak chapters using chapter practice
- Solve 10-15 questions per day per chapter
- Review all mistakes

**Week 5-6: PYQ Practice**
- Solve past year papers under timed conditions
- Aim for 60%+ accuracy
- Track trending topics

**Week 7-8: Revision**
- Use flashcards for vocabulary
- Re-solve previous mistakes
- Take 1-2 full mocks weekly

**Week 9-10: Final Sprint**
- Mock tests every 2-3 days
- Analyze weak topics
- Practice time management

### Best Practices

✅ **Time management:** Practice with 60-minute timer
✅ **Strategic guessing:** Don't leave blanks if >60% confident
✅ **Review mistakes:** Spend 10 minutes understanding each wrong answer
✅ **Track trends:** Monitor which chapters/topics are weak
✅ **Consistency:** Practice 1 hour daily minimum
✅ **Full mocks:** Take at least 10 complete mocks before exam
✅ **Note-taking:** Keep a separate notebook for formulas & rules
✅ **Peer discussion:** Discuss confusing topics with friends

## Support & Troubleshooting

### Issue: Application won't load
**Solution:** 
- Clear browser cache (Ctrl+Shift+Delete)
- Try a different browser
- Ensure file path is correct
- Check file isn't corrupted

### Issue: Login not working
**Solution:**
- Clear localStorage (developer tools > Application > Clear Storage)
- Ensure caps lock is off
- Create new account instead of using default
- Try incognito/private window

### Issue: Scores not saving
**Solution:**
- Check browser allows localStorage
- Disable privacy mode (may block storage)
- Ensure you submit test properly (wait for confirmation)
- Check available disk space

### Issue: Poor performance / lag
**Solution:**
- Close unnecessary browser tabs
- Clear browser cache
- Increase RAM allocation to browser
- Try desktop version instead of mobile

### Issue: Explanations not showing
**Solution:**
- Click "SOLUTIONS" button after test submission
- Wait for page to fully load
- Refresh page and retry
- Check internet connectivity (for any CDN resources)
